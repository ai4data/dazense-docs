## Usage Analytics

* Messages sent by users to the chat
* Total tokens consumed
* Total cost of tokens consumed

Users can filter results by specific model and adjust granularity to day, week, or month views.

## Feedbacks Monitoring

This section displays all user feedback collected during message threads. The Feedbacks table contains:

| Column | Description |
|--------|-------------|
| **Vote** | Thumbs up or thumbs down indicators |
| **User** | Feedback provider identification |
| **Message** | Preview of the rated agent response |
| **Reason** | User-supplied explanation (optional) |
| **Date** | Feedback timestamp |
