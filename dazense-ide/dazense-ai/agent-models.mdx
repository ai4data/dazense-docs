## Available LLM Providers and Models

**Claude (Anthropic)**
- Claude 3.7 Sonnet
- Claude 4 Sonnet
- Claude 4.5 Sonnet

**GPT (OpenAI)**
- GPT-5
- GPT-5.1
- gpt-oss

**Gemini (Google)**
- Gemini 2.5 Flash
- Gemini 2.5 Pro
- Gemini 3 Pro

**AWS Bedrock**
- Various models available through AWS Bedrock (requires AWS credentials or API keys)

## Setup Instructions

1. Navigate to dazense settings in the top right corner
2. Select the "Agent" section, then "Model Providers"
3. Enable your preferred LLM providers (Claude is strongly recommended)
4. Switch models directly within chat using the model selector in the header

## AWS Bedrock Integration

The platform supports AWS Bedrock models through either API Keys or AWS Credentials for authentication.

## Custom API Keys (Enterprise Feature)

Using your own API keys for LLM models is available only in Enterprise plans. This capability supports OpenAI, Anthropic, Bedrock, and Gemini keys, helping organizations maintain security compliance. Companies interested in this feature should contact the dazense team directly.
